# Changelog

## m2.0

- Procedural floor generation — 4-6 rooms per run via room-level WFC
- Three room types: corridor, arena, junction
- Two new enemy types: aggressor drone (charge attack) and sentry turret (burst fire)
- Environmental hazards: laser tripwires, sweeping beams, malfunctioning vents
- Vendor NPC with proximity reactions
- Loot quality tiers (salvaged, standard) with visual glow distinction
- Room clearing and door transition system
- Placeholder sound effects for combat feedback
- Debug god mode (G key) for playtesting

## m1.0

Initial playable build. Corridor combat with isometric rendering.

# Star Atlas: Null Session

A browser-based isometric action RPG set in the Star Atlas universe.

You are a Ghost. No name. No faction. No record in any system that should
know you exist. The station is corrupted. Fight your way through.

## Play Now

[Play in Browser](https://ttdonovan.github.io/arcade-null-session)

No install required.
